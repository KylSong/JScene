JScene.js

This library covers the basic methods needed to render 3D objects, including graphics display, basic optics, illumination and reflectance models, radiometry, curves and surfaces, and ray tracing. This library works with canvas to build the viewport in HTML and provide functions to make raster images that feed into this viewport. The features mainly focus on creating scenes (3D world), material, add/edit/transform the objects, edit camera, and scene properties.
